# Basic dApp NFT Landing Page

This landing page is created for the purpose of tracking the minting the date of NFTs. 

## Instructions

- Clone or download zipped code
- Update `index.html` file
  - \<title\>
  - favicon images
  - Logo
  - Social Links
  - Countdown `data-date`
  - Heading and paragraph text
- Update `style.css` file
  - Fonts
  - Colors
